# Instagram運用管理ツール セットアップ＆操作ガイド

## はじめに

このツールは、Googleスプレッドシート上でInstagram・LINE・議事録を一元管理するシステムです。
すべての操作はスプレッドシートのメニュー「📊 IG管理ツール」からボタンを押すだけで行えます。


---


## 1. 初期セットアップ（初回のみ）

### 1-1. Gemini APIキーの取得

AIによる考察生成・スクショ読み取りに必要です。

1. https://aistudio.google.com/apikey にアクセス
2. Googleアカウントでログイン
3. 「APIキーを作成」をクリック
4. 表示されたキー（AIza...で始まる文字列）をコピー

※ 無料枠で月間十分に利用できます。


### 1-2. スプレッドシートの初期設定

1. スプレッドシートを開く
2. メニュー「📊 IG管理ツール」→「⚙️ 設定」→「🚀 初期セットアップ（初回のみ）」
3. APIキーの入力を求められるので、1-1で取得したキーを貼り付けてOK
4. 自動でデータ更新・グラフ作成・書式設定が行われます

※ メニューが表示されない場合はページをF5で再読み込みしてください。

## 2. Instagramデータの入力方法

データの入力方法は2つあります。

### 方法A: スクショから自動読み取り（推奨）

スマホでInstagramのインサイト画面をスクリーンショットし、Googleドライブに入れるだけで自動的にデータを読み取ります。


#### 投稿データの取得手順

1. スマホでIGアプリを開く
2. プロフィール → インサイト → コンテンツ → 投稿一覧の画面を表示
3. 上から順にスクロールしながらスクリーンショットを2〜3枚撮影
4. 撮影したスクショをGoogleドライブの以下のフォルダに保存
   → 「IGインサイトスクショ」→「投稿」フォルダ
5. スプレッドシートで「📊 IG管理ツール」→「📸 投稿スクショ読み取り」
6. 自動で読み取りが行われ、①投稿管理シートにデータが書き込まれます

※ 処理済みのスクショは自動で「処理済み」フォルダに移動します。


#### アカウントインサイトの取得手順

1. IGアプリ → プロフィール → インサイト → 概要画面を表示
2. 以下の画面を上から順にスクロールしながらスクリーンショット（約4枚）
   - 閲覧数・フォロワー内外の割合
   - オーディエンス（年齢層）
   - プロフィールアクティビティ（プロフアクセス・リンクタップ）
   - フォロワー数・フォロワー増減
3. スクショをGoogleドライブの以下のフォルダに保存
   → 「IGインサイトスクショ」→「アカウント」フォルダ
4. スプレッドシートで「📊 IG管理ツール」→「📸 アカウントスクショ読み取り」
5. 自動で読み取りが行われ、②月次インサイトシートにデータが書き込まれます

※ 読み取れなかった項目がある場合、完了メッセージに警告が表示されます。
   その場合は②シートを確認し、手動で補完してください。


### 方法B: 手動入力

①投稿管理シート・②月次インサイトシートに直接数値を入力することもできます。

## 3. 月次レポート作成の流れ

毎月のレポート作成は以下の順番で行います。

### ステップ1: データ入力
上記「2. Instagramデータの入力方法」でデータを入力

### ステップ2: 全データ更新
- ③自動集計シートのB3セルで対象月を選択
- 「📊 IG管理ツール」→「🔧 全データ更新」
- ③に集計結果が自動反映、⑤レポート出力も自動構築されます

### ステップ3: AI考察生成
- 「📊 IG管理ツール」→「🤖 AI考察を生成」
- ④考察・コメントシートに4セクション（傾向・良い点・改善点・次月施策）が自動生成
- ⑤レポート出力にもAI考察が反映されます

### ステップ4: 確認・手動修正
- ④の内容を確認し、必要に応じて手動で修正
- ⑤レポート出力シートの内容も確認

### ステップ5: PDF出力
- 「📊 IG管理ツール」→「📄 レポートをPDF出力」
- Driveの「IGレポート」フォルダにPDFが保存されます

## 4. LINE分析の使い方

### CSVのダウンロード方法

1. LINE公式アカウント管理画面にログイン（ https://manager.line.biz/ ）
2. 「分析」→「統計情報をダウンロード」
3. 以下の2つをダウンロード
   - 「友だち」→「概要」（friend_overview_YYYYMMDD_YYYYMMDD.csv）
   - 「メッセージ通数」（message_overview_YYYYMMDD_YYYYMMDD.csv）

### 取り込み手順

1. ダウンロードしたCSVをGoogleドライブの「LINE_CSV」フォルダに保存
2. スプレッドシートで「📊 IG管理ツール」→「📱 LINE CSVを取り込み」
3. 日次データが自動で月次に集計され、⑦LINE分析シートに書き込まれます

### 取り込まれるデータ

| 項目 | 元データ |
|------|---------|
| 友だち数 | 月末時点の友だち数 |
| ターゲットリーチ | 月末時点のターゲットリーチ数 |
| 新規友だち | 月初〜月末の友だち増加数 |
| 新規ブロック | 月初〜月末のブロック増加数 |
| 自動応答〜API返信 | 各メッセージ種別の月間合計 |

### LINE AI考察
- 「📊 IG管理ツール」→「📱 LINE AI考察を生成」
- ⑦シートのデータに基づいてAIが傾向と改善点を自動生成します

※ 開封率・クリック率はメッセージ配信を行っている場合のみCSVに含まれます。

## 5. 議事録の使い方

打ち合わせの録音ファイルから、自動で文字起こし＋議事録フォーマットに整理します。

### 手順

1. 打ち合わせを録音（iPhoneのボイスメモ等）
2. 録音ファイル（m4a/mp3/wav）をGoogleドライブの「議事録」→「録音」フォルダに保存
3. スプレッドシートで「📊 IG管理ツール」→「📝 議事録を処理」
4. 録音ファイルの一覧が表示されるので、処理する番号を入力
5. クライアント名・日付を入力
6. 処理開始（バックグラウンドで10〜15分かかります）
7. 完了するとDriveの「議事録」→「出力」フォルダに議事録が保存されます

※ 初回実行時にGitHub Personal Access Tokenの入力を求められます。
   一度入力すれば以降は不要です。

### 議事録のフォルダ構成

```
議事録/
├── 録音/       ← ここに録音ファイルを入れる
├── 処理済み/   ← 処理後の録音が自動移動
├── 文字起こし/ ← 生テキスト（バックアップ）
└── 出力/       ← 完成した議事録
```

## 6. 新規クライアント追加

1. 「📊 IG管理ツール」→「📋 テンプレートをコピー」
2. クライアント名を入力
3. 同じフォルダに「IG管理_クライアント名」として新しいスプシが作成
4. すべてのデータがクリアされた状態で、すぐに使い始められます

※ コピー先でもメニューが使えます。ただしAPIキーはコピー先で再設定が必要です。


---


## 7. Driveフォルダ構成

```
プロジェクトフォルダ/
├── Instagram管理ツール（スプレッドシート本体）
├── IGレポート/              ← PDF出力先
├── IGインサイトスクショ/
│   ├── 投稿/               ← 投稿インサイトのスクショを入れる
│   └── アカウント/          ← アカウントインサイトのスクショを入れる
├── LINE_CSV/               ← LINE管理画面のCSVを入れる
└── 議事録/
    ├── 録音/               ← 録音ファイルを入れる
    └── 出力/               ← 完成した議事録
```

## 8. トラブルシューティング

### メニューが表示されない
→ スプレッドシートをF5で再読み込みしてください。

### 「APIキー未設定」と表示される
→ 「⚙️ 設定」→「🔑 Gemini APIキーを設定」からキーを入力してください。

### AI考察が生成されない
→ 「⚙️ 設定」→「🔍 APIキー動作テスト」でAPI接続を確認してください。

### スクショ読み取りが失敗する
→ 画像が暗い、解像度が低い可能性があります。明るい状態で撮り直してください。
→ 対応形式: PNG / JPG / JPEG / WebP

### LINE CSVが読み込まれない
→ 「LINE_CSV」フォルダに直接CSVを配置してください（サブフォルダ内は対象外）。

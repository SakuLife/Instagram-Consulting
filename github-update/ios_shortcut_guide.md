# iPhone用 IGインサイト自動スクショ ショートカット設定ガイド

## 概要

iPhoneの「ショートカット」アプリを使って、
IGインサイト画面のスクショ → Googleドライブ保存を簡略化します。

---

## 事前準備

1. iPhoneに「ショートカット」アプリがインストールされていること（標準搭載）
2. iPhoneに「Googleドライブ」アプリがインストールされていること
3. Googleドライブアプリにログイン済みであること

---

## ショートカット① 「IG投稿スクショ」の作り方

### 手順

1. 「ショートカット」アプリを開く
2. 右上の「+」で新規作成
3. 名前を「IG投稿スクショ」に変更
4. 以下のアクションを順番に追加:

```
アクション1: スクリーンショットを撮る
アクション2: 名前を変更 → 「post_[現在の日付].png」
アクション3: Googleドライブに保存
  → 保存先: IGインサイトスクショ > 投稿
  → 上書き: しない
```

### 設定詳細

「Googleドライブに保存」の設定:
- アカウント: ログイン済みGoogleアカウント
- フォルダパス: IGインサイトスクショ/投稿
- ファイル名: post_1.png（手動で変更可）

---

## ショートカット② 「IGアカウントスクショ」の作り方

上記と同様に作成。保存先だけ変更:
- フォルダパス: IGインサイトスクショ/アカウント
- ファイル名: account_1.png

---

## 使い方（毎月のルーティン）

### 投稿インサイト（2〜3枚）

1. IGアプリ → プロフィール → インサイト → コンテンツ → 投稿一覧
2. コントロールセンターまたはホーム画面からショートカット「IG投稿スクショ」を実行
3. 画面をスクロール
4. もう一度ショートカットを実行
5. 必要に応じて繰り返し（2〜3回）
6. スプレッドシートで「📸 投稿スクショ読み取り」ボタンを押す

### アカウントインサイト（4枚）

1. IGアプリ → プロフィール → インサイト → 概要
2. ショートカット「IGアカウントスクショ」を実行
3. 画面をスクロール → 実行 を4回繰り返し
4. スプレッドシートで「📸 アカウントスクショ読み取り」ボタンを押す

---

## より簡単にする方法

### 方法A: ウィジェットに追加
1. ホーム画面を長押し → ウィジェット追加 → ショートカット
2. 「IG投稿スクショ」「IGアカウントスクショ」を配置
3. ホーム画面から1タップで実行可能

### 方法B: 背面タップに割り当て（iPhone 8以降）
1. 設定 → アクセシビリティ → タッチ → 背面タップ
2. ダブルタップ or トリプルタップにショートカットを割り当て
3. IGのインサイト画面を見ながらiPhoneの背面をタップするだけ

### 方法C: Siriで実行
「Hey Siri、IG投稿スクショ」と言うだけで実行可能

---

## 補足

- スクショは自動でDriveにアップされるので、PCでの操作は不要
- スプレッドシートの「📸 読み取り」ボタンはPC・スマホどちらからでも押せる
- スクショの順番はファイル名で並ぶので、post_1, post_2...の順に撮ってください
